首先回顾一下proximal gradient descent。（这一部分也忘了）
\[ \min \limits_{x} g(x) + h(x) \tag{1} \],
其中$ g, h $都是凸函数， $ g $可微， $ h $很简单，那么：
\[ prox_{t} = \arg\min\limits_{x} \frac{1}{2t} | x - z |_{2}^{2} + h(x) \tag{2} \]
Proximal gradient descent : let $ x^{(0)} \in \mathbf{R}^n $, repeat:
\[ x^{(k)} = prox_{t_k} (x^{(k-1)} - t_k \nabla g(x^{(k-1)})), k=1, 2, 3, \cdots \tag{3}\]
